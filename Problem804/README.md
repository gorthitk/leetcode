### Problem 804. Unique Morse Code Words 

### Description
    International Morse Code defines a standard encoding where each letter is mapped to a series of dots and dashes, 
    as follows: "a" maps to ".-", "b" maps to "-...", "c" maps to "-.-.", and so on.
